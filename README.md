# YouTube Clone

This project is a static front-end clone of the YouTube homepage built with HTML and CSS. It recreates the overall look and feel of YouTube's main interface, including a top navigation bar, sidebar, and a video grid with thumbnails, channel details, and video statistics.

## Live Demo

Visit: https://clone-y0utube.netlify.app/

## Features

- Responsive homepage-style layout
- Sidebar navigation with icons
- Top bar with search, upload, notifications, and profile section
- Video cards with thumbnails and metadata
- Styled tooltips and hover effects

## Project Structure

- index.html - Main page structure
- genral.css - Shared/global styles
- video.css - Video card styling
- top-bar.css - Top navigation bar styles
- side-bar.css - Sidebar styles
- popup.css - Tooltip and popup styles
- menubar/ - Icons and UI assets for the top bar
- navbar/ - Sidebar icons
- video-thumbnail/ - Thumbnail images
- profile-picture/ - Channel profile images
- channel-pic/ - User/profile image assets

## How to View

1. Open the project folder in your browser.
2. Double-click index.html to view the page, or use a simple local server.



## Notes

This is a frontend UI clone and does not include real video playback, dynamic search, or backend functionality.
